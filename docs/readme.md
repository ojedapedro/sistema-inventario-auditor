# Sistema de Control de Inventario

Sistema web profesional para auditoría de inventarios que permite comparar el inventario teórico con el físico mediante escaneo de códigos de barras.

## Características

- Carga de inventario teórico desde archivos Excel
- Escaneo de códigos de barras para el inventario físico
- Generación de informes PDF con discrepancias
- Historial de los últimos 5 inventarios
- Interfaz moderna y responsive
